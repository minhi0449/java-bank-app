## ✓ 핵심 기능
**✅ 계좌 생성
	•	계좌번호, 예금주, 초기 잔액 정보를 입력하여 새 계좌 생성

✅ 계좌 목록 조회
	•	모든 계좌 정보 확인

✅ 입금 기능
	•	계좌번호를 입력하여 금액을 입금

✅ 출금 기능
	•	계좌번호를 입력하여 금액을 출금**
